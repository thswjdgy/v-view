v-view
AI 면접 코칭 앱 · 중간발표
손정협 | 2026-06-08
FLUTTER FIREBASE OPENAI API ML KIT RIVERPOD

혼자 하는 면접 준비는 실전감과 객관적 피드백이 부족하다.

v-view는 AI 질문 생성과 시선 분석으로 실전형 면접 훈련을 제공한다.

답변 내용뿐 아니라 시선 처리와 긴장 습관까지 개선할 수 있다.

02 / 07
혼자 하는 면접 연습엔
실전감도, 피드백도 없다
기존 면접 준비 방법은 비용, 주관성, 방향 부재라는 한계를 안고 있습니다.
v-view는 이 세 가지 문제를 AI와 온디바이스 분석으로 정면 돌파합니다.
기존 방법의 한계
면접 코치
건당 5~10만원 — 비용 부담이 크고 접근성이 낮다
친구 도움
불편하고 주관적 — 일관된 기준을 기대하기 어렵다
혼자 녹화
무엇을 개선해야 할지 알 수 없는 막막함
유튜브 강의
일방향 콘텐츠 — 내 습관은 그대로 남는다
v-view의 해결
AI 맞춤 질문
직종·자소서 기반으로 개인화된 질문을 자동 생성
OpenAI API가 직종·자소서 기반으로 기본 3개 + 꼬리 질문 자동 생성
실시간 시선 분석
카메라로 시선 처리를 온디바이스에서 즉시 분석
ML Kit 온디바이스 처리 — 영상이 서버로 전송되지 않음
즉시 피드백 리포트
세션 종료 후 개선 TOP3를 포함한 상세 리포트 제공
세션 종료 후 응시율, 분산 횟수, 개선 TOP3 즉시 제공
프라이버시 보장
원본 영상 서버 전송 없음 — 완전한 온디바이스 처리
원본 영상 저장 없음 — Hive로 기기 내 로컬 저장만

04 / 07
시스템 아키텍처
단방향 의존성 원칙을 엄격히 준수합니다. UI 레이어는 Data 레이어를 직접 참조하지 않으며, 각 계층은 명확히 분리된 책임을 가집니다.

단방향 의존성 — UI는 Data를 직접 참조하지 않습니다. 모든 데이터 흐름은 Application 및 Domain 레이어를 통해서만 이루어지며, 이는 유지보수성과 테스트 용이성을 크게 향상시킵니다.

05 / 07
구현된 핵심 기능
AI 맞춤 질문 생성
OpenAI API를 활해 직종과 자기소개서를 기반으로 개인화된 질문을 실시간 생성합니다. 꼬리 질문까지 자동 생성하여 실전 면접에 가까운 경험을 제공합니다.
MVP: 기본 질문 3개 + 꼬리 질문 자동 생성
실시간 시선 분석
ML Kit 온디바이스 처리로 시선 이탈을 실시간 감지합니다. 연속 1초 이상 이탈 시 기록되며 서버로 영상이 전송되지 않아 프라이버시가 완전히 보호됩니다.
MVP: 응시율(%) + 분산 횟수/시간 측정
세션 피드백 리포트
세션 종료 즉시 응시율, 개선 TOP3, 히스토리를 포함한 상세 리포트를 제공합니다. 반복 훈련을 통한 성장 곡선을 시각적으로 확인할 수 있습니다.
MVP: 시선 지표 + AI 피드백 + 개선 TOP3
Firebase 인증
안전한 회원가입·로그인 시스템과 클라우드 동기화를 지원합니다. 기기를 바꿔도 면접 히스토리와 설정이 그대로 유지됩니다.
MVP: 회원가입/로그인 + 클라우드 동기화 + 기기 간 히스토리 유지
