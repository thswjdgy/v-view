# 비전 & 목표

## 한 줄 비전

> 취업 준비생이 혼자서도 AI와 함께 실전 면접을 연습하고, 시선·답변 피드백으로 스스로 성장할 수 있는 앱

---

## 문제 정의

| 문제 | 현황 |
|---|---|
| 면접 코칭 비용 | 1회 10~30만원, 반복 불가 |
| 시선 처리 피드백 부재 | 거울 연습은 실제 면접과 다름 |
| 연습 기록 관리 어려움 | 메모·녹음 파편화 |
| 개인정보 유출 우려 | 영상 저장 서비스에 대한 거부감 |

---

## 핵심 가치 제안

1. **무제한 AI 맞춤 질문** — 직종·자기소개서 기반으로 매번 다른 질문
2. **실시간 시선 분석** — 카메라로 응시율·분산 횟수 자동 측정
3. **로컬 우선 개인정보 보호** — 원본 영상 미저장, 기기 내 데이터만
4. **회차별 성장 추이** — 최근 5회 응시율 트렌드 차트

---

## 목표 사용자

**1순위**: 취업 준비 중인 대학생·신입 지원자  
**2순위**: 이직을 준비하는 3년 이하 직장인  
**공통 특징**: 혼자 연습하고 싶지만 피드백을 받을 방법이 없는 사람

---

## 성공 지표 (MVP 기준)

| 지표 | 목표 |
|---|---|
| 면접 세션 완주율 | 1회 이상 완료 가능 |
| 시선 분석 정확도 | 정면 응시 판별 정상 동작 |
| 리포트 생성 | AI 성공 + fallback 모두 동작 |
| 앱 실행 | `git clone` 후 단일 명령으로 빌드 |

---

## 프로젝트 범위

### MVP (Must)
- AI 맞춤 질문 생성 (OpenAI gpt-4o-mini)
- 실시간 시선 분석 (ML Kit)
- 피드백 리포트 (시선 지표 + AI 개선 포인트)
- 면접 기록 관리 (히스토리 저장·삭제)
- 세션 설정 (유형 선택, 입력 재사용)
- 권한·오류 처리

### 제외 (Won't — MVP)
- 서버 저장 / 클라우드 동기화
- 소셜 기능 (공유, 랭킹)
- 음성 인식 답변
